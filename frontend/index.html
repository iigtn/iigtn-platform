このサイトのハイライト

本サイト自体が「運用中の AWS サーバレス構成」のサンプルです

☁️ AWS サーバレス

S3 + CloudFront (OAC) + Lambda + API Gateway + DynamoDB + SES + CloudWatch + SNS + Budgets。

🛠 Terraform IaC

5 モジュール (network_dns / frontend_cdn / backend_api / ci_oidc / observability) + S3 backend + DDB Lock。

🔁 静的キーレス CI/CD

GitHub Actions OIDC で IAM Role を一時 AssumeRole。長期アクセスキーを GitHub Secrets に置かない設計。

🔐 最小権限 + 観測

OAC で S3 を非公開維持、Public Access Block で構造的に公開不能化。CloudWatch Alarms + SNS Email + Budgets で異常検知。
