hojokin-db — 補助金検索データベース →

業種・地域・種別で絞り込める日本の補助金/助成金検索サイト。Next.js 16 (static export) + S3 + CloudFront (OAC) でフロントを配信、DynamoDB に補助金データを格納し、 Lambda + EventBridge + Amazon Bedrock の自動クロール&AI 整形 → CodeBuild の自動再ビルドで毎日更新する仕組みを構築しています。 Terraform で 25 リソース全部を IaC 化、月額 5,500〜10,500 円で運用中です。

autoST — 自動売買リサーチ＆運用フレームワーク →

  • DynamoDB
  • Terraform
  • Backtesting
  • kabu STATION API
