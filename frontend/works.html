hojokin-db — AWS サーバレス × 自動更新型データベースサイト →

Next.js 16 (SSG) を S3 + CloudFront (OAC) で配信し、DynamoDB に補助金データを格納する構成。 EventBridge から起動する Lambda + Amazon Bedrock で毎日 02:00 にソース巡回 → AI 整形、03:00 に CodeBuild で DDB→JSON→next build→S3 sync→CloudFront /* 無効化までを無人で回します。 Terraform で 25 リソース全部を IaC 化、月額 5,500〜10,500 円で運用中です。

AI えらびかた — Next.js × AWS Amplify Hosting メディア基盤 →

hojinka.com — 静的サイト × OIDC 自動デプロイ基盤 →

AWS S3 (private) + CloudFront (OAC) + ACM + Route 53 で配信する静的メディア基盤。 GitHub Actions OIDC で AssumeRoleWithWebIdentity による静的キーレス自動デプロイを実装し、 main ブランチ push のみで S3 sync + CloudFront 無効化までを完結させています。 法人化シミュレーターは Vanilla JS でブラウザ完結、サーバ・API なしで対話的なツールを提供。 全リソースを Terraform で IaC 化しています。

DocChain LP — Next.js Static × API Gateway/Lambda/DynamoDB 統合 →
